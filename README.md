# Learn-Python-With-Single-File
This file has many Python exercises to practice with. Everything to learn in introductory and object oriented Python is in here. 

The 19 projects included in this file are: simple calculator, weight converter, temperature converter, interest calculator, countdown timer, shopping cart program, quiz game, concession stand, number guessing game, rock/paper/scissors game, dice roller, credit card validator, slot machine, hangman game, alarm clock, digital clock, stop watch, weather app, music app.
